# **App Name**: AI Codebase Explainer & Debugger

## Core Features:

- Flexible Code Input: Allows users to upload a project ZIP file, provide a GitHub repository URL, or paste source code directly for analysis.
- AI Project Overview: Utilizes generative AI to create a concise, high-level summary of the codebase's purpose and functionality.
- AI Line-by-Line Explanations: Leverages generative AI as a tool to break down complex code snippets, explaining each line's logic and purpose in plain language.
- AI Debugging Assistant: Uses generative AI as a tool to identify potential bugs, suggest fixes, and provide explanations for common coding errors.
- Multilingual Code Analysis: Backend architecture designed to support code analysis and explanation for multiple programming languages including Python, Java, C, C++, JavaScript, and TypeScript.
- Intuitive User Interface: Provides a clean, beginner-friendly interface with dedicated sections for input, codebase overview, detailed explanations, and debugging results, including loading states and error handling.

## Style Guidelines:

- Color scheme: Dark theme, suggesting a technical environment common in development tools.
- Primary color: A deep, professional blue (#3636BF) to convey intelligence and reliability. (HSL: 240, 70%, 45%).
- Background color: A subtle, dark blue-grey (#1A1A20) derived from the primary hue, ensuring focus on content. (HSL: 240, 15%, 12%).
- Accent color: A vibrant cyan (#1EC9FF) that stands out for interactive elements and highlights, bringing a modern and energetic feel. (HSL: 210, 90%, 60%).
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired appearance, suitable for headings.
- Body font: 'Inter' (sans-serif) for its excellent readability and neutral, objective aesthetic, perfect for code explanations.
- Code font: 'Source Code Pro' (monospace sans-serif) for clear and consistent display of code snippets.
- Utilize minimalist, crisp line icons to maintain a clean and functional aesthetic that complements the modern design.
- Structured, multi-panel layout that logically separates input, overview, explanations, and debugging sections, ensuring a clear information hierarchy and responsiveness across devices.
- Incorporate subtle, functional animations for loading states, content transitions, and feedback, enhancing the user experience without distraction.
